
# Console CRUD App (MySQL & SQLite)

Простая консольная утилита на C# (.NET) для управления пользователями (базовые CRUD-операции). Программа демонстрирует работу с абстрактными классами `DbConnection`, `DbCommand` и `DbDataReader`, что позволяет динамически переключаться между двумя СУБД: **MySQL** и **SQLite**.

---

### Особенности проекта

* **Мульти-СУБД**: поддержка автономной базы SQLite и выделенного сервера MySQL.
* **Универсальный код**: использование фабричных методов ADO.NET для независимости от конкретного провайдера данных.
* **Автоматическая инициализация**: скрипты проверки и генерации таблиц учитывают диалекты SQL (например, синтаксис автоинкремента).

### Конфигурация (`appsettings.json`)

Перед запуском убедитесь, что в корневой директории проекта присутствует файл конфигурации со строками подключения. Пример структуры:

```json
{
  "ConnectionStrings": {
    "csMySql": "Server=localhost;Database=testdb;Uid=root;Pwd=your_password;",
    "csSqlite": "Data Source=users.db;"
  }
}
```

---

### Быстрый старт

1. **Восстановите зависимости** проекта NuGet:
   * `MySql.Data`
   * `Microsoft.Data.Sqlite`
   * `Microsoft.Extensions.Configuration.Json`

2. **Запустите проект** через IDE или CLI:
   ```bash
   dotnet run
   ```

3. **Выберите СУБД** вводом цифры `1` или `2` при запуске приложения и используйте текстовое меню для управления пользователями.
